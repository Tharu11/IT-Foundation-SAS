DV500_56_PAS On Cloud Models
Possible Answer Sheet


Q1. What is an example of cloud capability?
Ans. An example of cloud capability would be the deployment of new applications. 


Q2. Where can an application be deployed?
Ans. An application can be deployed on a web server, database server, application server.


Q3. To deploy the application, you can use Data centers that are located in ____________.
Ans. US, Europe, Asia.


Q4. Can you move those application instances from one data center to another where you are effectively following the sun?
Ans. Yes, application instances can be moved from one data center to another where we are effectively following the sun.


Q5. What is an efficient way to deploy an application?
Ans. An efficient way to deploy an application would be the data center that is closest to us.


Q6.  What do some organizations do instead of going to a third party for cloud service?
Ans. Some organizations purchase their own equipment and build their own cloud within their own data center instead of going to a third party for cloud service.


Q7. What is a private cloud?
Ans. A private cloud is when a company decides to build their own cloud with their own equipment. This way only the company has access to and is not designed to be shared with anyone else.


Q8. Cloud services provided by Amazon, Microsoft, Rackspace, and others can be configured as what?
Ans. Cloud services provided by Amazon, Microsoft, Rackspace, and others can be configured as public cloud.


Q9. What is the combination of using both a private and public cloud?
Ans. The combination of using both a private and public cloud can be seen as a hybrid cloud where both private and public cloud service is used.


Q10. What are the advantages of having a private cloud?
Ans. The advantage of having a private cloud would be organizations with similar goals can chip in to purchase their own cloud equipment and share them so just the one organization doesn’t have to purchase all of the equipment on their own.


Q11. What does an organization build that include a database, web server and other devices?
Ans. An organization might build their own application that includes a database, web server, and other devices.


Q12. A company may only need the _______________ to be able to load the software and deploy these cloud instances.
Ans. Raw infrastructure.


Q13. If an organization only needs CPU storage and a network to be able to deploy the application instance, then they want to use ____________________as their cloud deployment model. 
Ans. Infrastructure.


Q14. What does HaaS stand for?
Ans. HaaS stands for Hardware as a Service.


Q15. Who is responsible for a large amount of implementation in a cloud model?
Ans. For a large amount of implementation in a cloud model the end user is responsible.


Q16. What does a cloud provider provide?
Ans.  A cloud provider is effectively providing the hardware to run the application and in cases, perhaps even the operating system.


Q17. What is an example of IaaS deployment?
Ans. An example of IaaS (Infrastructure as a service) deployment would be a company that provides web services. They would provide us with the server to be able to load the website.


Q18. What is the one main benefit the end user has in SaaS?
Ans.  The one main benefit the end user has in SaaS (Software as a service) is the end user doesn’t have to deal with anything relating to the application or the operating system.


Q19. What does SaaS stand for?
Ans.SaaS stands for Software as a Service.


Q20. What isn’t required for any type of software?
Ans. There is no local installation required for any type of software.


Q21. The end user can simply log in and manage an email distribution list, log in and manage their payroll or any other type of software as a ______________.
Ans. A service implementation.


Q22. Who is responsible for maintaining the application, storing the data, and keeping all of that information safe?
Ans. The software as a service provider is the one responsible for maintaining the application, storing the data, and keeping all of the information safe.


Q23. Google mail is an example of _____________.
Ans.Google mail is an example of software as a service.


Q24. Are you responsible for maintaining the google mail?
Ans. No, Google is responsible for maintaining the google mail.


Q25. What does Google do with the email system?
Ans. Google provides us with the entire email system with simply username and password.


Q26. A good example of Saas is ______________.
Ans. A good example of Saas is Microsoft 365.


Q27. The middle ground between Iaas and Saas is __________________.
Ans. PaaS (Platform as a service) 


Q28. What is PaaS?
Ans. PaaS can be seen as someone else providing us with a platform that we can use to build our own applications. 


Q29. What does the PaaS provider do?
Ans. The PaaS provider provides us with all of the building that we would need to put together every piece of that application. 


Q30. ______________has an extensive platform as a service offering that gives you complete customization over the applications that you can easily deploy on their platform.
Ans. Salesforce.


Q31. If you were planning to deploy the applications in your own data center on your own hardware using applications that you yourself have written, then you’re probably using an ____________where everything in that model is managed by the client.
Ans. On-premises model.
